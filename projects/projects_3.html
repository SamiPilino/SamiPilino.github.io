Process Timer

This is a pomodoro-style timer that I use to help organize and track how I use my time.

Changelog

  • Designed custom list of setTimeout's to loop through
  • Commented list loop out and re-architect-ed a countdown-timer I had in a way that made it much more intuitive to reason about and attach events and logic to
  • Implemented user experience matched how I wanted the timer to function

Roadmap

  • Music
  • I need one sound to play during the sections and another sound to signal when it's time to write the summary.
  • Configuration UI
  • A configuration UI would help usability for others, but I'm the only one using it now, so I'll just hard-code configurations for now.
  • Integrations
  • Still trying to figure out how to connect it with other project ideas such as the "schedules", "plans", "kanban boards", "AIOps", etc. that I have in the works.
